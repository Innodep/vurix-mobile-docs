# Sign In

<p align = "left">
<Image src="https://github.com/Innodep/vurix-mobile-docs/blob/main/assets/SignIn.png?raw=true"/>
</p>

1. VURIX Mobile App을 실행한다.
	- My Page>Settings>Security에서 Auto Sign In을 Enable한다면 App 실행 시 자동 로그인된다.
2. Tap **Sign In** to access the app.
	- 최초 로그인 시 Server, Port, ID, Password 정보를 입력한다.
3. 로그인 이력이 있으면 선택하여 Sign In 할 수 있다.
	- 다른 계정으로 로그인하면 이력이 추가된다.
4. 불필요한 로그인 이력을 삭제할 수 있다.

>**Important Notes:**
- ID는 VURIX에서 로그인이 되는 계정을 부여 받거나 신규 사용자 등록을 해야 한다.
