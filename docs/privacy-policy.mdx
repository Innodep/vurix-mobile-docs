# Privacy Policy (English)

## 1. Collected Information

The VURIX app stores server connection details provided by the user for convenient login.

- **Stored Information**: IP address, Port number, User ID, Password
- Data received from the connected server is displayed within the app and is not transmitted externally.

## 2. Purpose of Data Usage

The collected information is used for:

- Providing seamless login functionality
- Connecting to the designated server to display video and data

## 3. Data Security and Protection

Stored connection details remain on the user's device and are encrypted for security.
